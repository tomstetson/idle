## Track 4: Experiential Enhancements — PLANNED (Future Sprint)

These features need design decisions and user input. Planned but not implemented in v0.1.1.

### 4.1 Slash Command Discovery
**Current State:**
- `CommandPalette` component exists but is **web-only** (Platform check)
- Keyboard-triggered only — no visual affordance
- Categories: Sessions, Navigation, System, Developer, Recent Sessions
- `useAutocomplete` provides in-chat slash command suggestions
- `suggestionCommands.ts` has dynamic + static command descriptions

**Enhancement Plan:**
- Add visual trigger button in header or FAB for mobile users
- Show available commands in-chat as a help card on first session
- Add keyboard shortcut hints in palette UI
- Consider native mobile command palette (bottom sheet)

### 4.2 Session Details Pane
**Current State:**
- `session/[id]/info.tsx` shows rich metadata: host, path, OS, version, tools, commands, git status
- Session list shows: avatar, title, relative path, date grouping
- Active sessions show: status dots, thinking indicators, permission badges

**Enhancement Plan:**
- Surface status dots inline in session list (currently only in detail view)
- Add session duration / message count to list preview
- Add badges for "has MCP tools", "requires permission" in list items
- Quick actions on swipe (already exists for "Move to Top")

### 4.3 Session Icons / Visual Identity
**Current State:**
- Sessions use generic `Avatar` component with gradients (99 gradient images available)
- No per-session visual differentiation beyond the avatar

**Enhancement Plan:**
- Generate deterministic color/gradient from session tag (hash-based)
- Add project-type indicators (icon for web, CLI, API, etc.)
- Use machine OS icon as secondary indicator
- Consider auto-generated abstract art icons from session metadata

### 4.4 Additional Opportunities
- Friend feed improvements
- Server configuration validation (web + app)
- Session groups (name-only, drag to group) — from Phase 5 roadmap
- Hold-to-reorder with grip icon — from Phase 5 roadmap
